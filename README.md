\# Memristor VTEAM Model Simulation



Python implementation of a simple VTEAM-based memristor model for RRAM devices.



\## Features

\- Voltage sweep simulation

\- I-V hysteresis curve

\- State variable evolution



\## Installation



git clone https://github.com/SLD-Varma/memristor-vteam-model.git



cd memristor-vteam-model



pip install -r requirements.txt



\## Run Example



python examples/iv\_simulation.py



\## Model Parameters



R\_on : Low resistance state  

R\_off : High resistance state  

w : State variable (nm)  

d : Oxide thickness  



\## Example Output



\- I-V hysteresis loop

\- State variable evolution
